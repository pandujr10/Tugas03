Belajar CSS

Apa Itu CSS

CSS adalah singkatan dari "Cascading Style Sheets". Ini adalah bahasa stylesheet yang digunakan dalam pengembangan web untuk menggambarkan presentasi dan gaya visual halaman web yang ditulis dalam HTML (Hypertext Markup Language) atau XML (eXtensible Markup Language). CSS memungkinkan pengembang web untuk mengontrol bagaimana elemen pada halaman web ditampilkan,
termasuk tata letak, warna, font, spasi, dan banyak lagi.

Bagaimana Menggunakan CSS

Ada 3 Cara Menggunakan CSS:

  1. inline:Gaya diterapkan langsung ke elemen HTML individual menggunakan atribut "style".
  2. Gaya didefinisikan dalam bagian "head" dokumen HTML menggunakan tag.
  3. gaya didefinisikan dalam file CSS terpisah dengan ekstensi ".css", yang kemudian ditautkan ke dokumen HTML menggunakan.
